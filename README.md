## 🕹️ วิธีเล่น

### คีย์บอร์ด
| ปุ่ม | การทำงาน |
|------|---------|
| ← → ↑ ↓ หรือ WASD | เดิน / วิ่ง |
| Z หรือ X | เตะ |
| Space / ↑ | กระโดด (ด่าน Platform) |

### iPad / มือถือ
ใช้ปุ่มบนหน้าจอได้เลย! 👆
- **D-pad** ด้านซ้าย — เดินทุกทิศ
- **👊** ด้านขวา — เตะ
- **↑** ด้านขวา — กระโดด

> แนะนำให้หมุนหน้าจอ **แนวนอน (Landscape)** เพื่อประสบการณ์ที่ดีที่สุด
